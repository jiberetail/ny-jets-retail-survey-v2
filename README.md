# NY Jets Retail Survey V2

Interactive 9:16 Jets game-day services kiosk. This is a separate Version 2
experience and does not replace the original retail survey.

## Public Demo

[Open the NY Jets Retail Survey V2](https://jiberetail.github.io/ny-jets-retail-survey-v2/)

The public demo is deployed from the `jiberetail` GitHub organization. Public
project URLs must use an organization-owned `jiberetail.github.io` address and
must never use a personal account name.

## Development

```bash
pnpm install
pnpm run dev
pnpm run build
```
